# birc-bids

birc-bids provides a Docker/Singularity container with various BIDS conversion utilities.

## Installed Software

### Conversion

- [BIDScoin](https://github.com/Donders-Institute/bidscoin)
- [dcm2niix](https://github.com/rordenlab/dcm2niix)
- [Dcm2Bids](https://github.com/cbedetti/Dcm2Bids)
- [bidskit](https://github.com/jmtyszka/bidskit)
- [heudiconv](https://github.com/nipy/heudiconv)


### Utilities

- [bids-tools](https://github.com/robertoostenveld/bids-tools)
- [bids-validator](https://github.com/bids-standard/bids-validator)
- [pybids](https://github.com/bids-standard/pybids)
- [nipype](https://nipype.readthedocs.io/en/latest/)
